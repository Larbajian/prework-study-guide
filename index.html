Git

  • git statues: check what branch we are currently on
  • git checkout -b branch-name: creates a new branch
  • git add -A: signals to system that all changes made within branch will be saved at next commit
  • git commit -m "description of changes": saves changes made within branch. "-m" associates a message with your commit. Be sure to add a description for later reference.
  • git pull origin main: pulls branch modifications into local workspace. "Main" indicates which branch you're pulling from.
  • git push origin branch name: pushes changes made in local workspace to remote workspace. You can create a new remote branch or add changes to an existing branch.
